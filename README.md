### ⚙️ RISC-V Single-Cycle Processor *(In Progress)*
> VHDL · QuestaSim · Computer Architecture

Designing a complete **32-bit RISC-V single-cycle processor** in VHDL from scratch.
- Built core components: **ALU, Register File, Control Unit, Instruction & Data Memory**
- Supports **R-type, I-type, LW, SW, and Branch** instructions
- Verified with testbenches and waveform analysis in **QuestaSim**

### 🤖 [CyTank — Bluetooth Combat Robot](https://github.com/jppatel123/CYtank-cpre288-finalproject-)
> C · Tiva C TM4C123G · Arduino · UART · I2C · PyQt5

Full-stack combat robot — bare-metal firmware + live Python GUI.
- **Firmware in C** with UART, GPIO, interrupts, and servo for 180° radar scanning
- **Arduino ↔ Tiva C via I2C** to fire a KY-008 laser from the GUI shoot button
- **Python PyQt5 GUI** with live radar display, IMU tracking, and Bluetooth control
